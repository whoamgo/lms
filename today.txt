HR Panel Requirements
=================



1.Employee Master

	A.The Employee Master module should be created in the HR panel, similar to the one created in the Admin panel.

	B.HR should be able to create, view, edit, and manage employee details.

2.Attendance Management

	A.HR will create and manage employee attendance.

	B.The attendance management process should be the same as the Admin panel.

	C.Attendance records should be viewable, editable, and searchable.

3.Leave Management

	A. A Leave Management module should be created in the HR panel, similar to the Admin panel.
	
	B.When an employee applies for leave:

	C.The leave request should be visible in both HR and Admin panels.

	D.HR and Admin should be able to approve or reject the leave request.

	E.Employees should be able to view the status of their leave requests(Employees panel).

4.Contract Renewal Management

	A.Contract details should be saved when an employee is created.

	B.A list of all employee contracts should be available in the HR panel.

	C.HR should be able to renew employee contracts.

	D.Old contract history should also be maintained and viewable.

	E.If an employeeâ€™s contract is closed or expired, the employee should not be able to log in to the system.

5.Payroll Coordination

	A.The HR panel should provide a Payroll View.

	B.HR should be able to generate payslips and download them.

	C.Payroll search functionality should be available:

	D.Search by employee name

	E.Same search and filter options as in the Admin panel

6.Reports Master

 
	A.Reports functionality and structure should be the same as the Admin panel.

	B.HR should be able to generate and view various reports.
